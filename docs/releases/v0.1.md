# Maverything 0.1

First public test release of Maverything: an Everything-style instant file search app for macOS.

## Highlights

- Native macOS app built with SwiftUI and AppKit
- Instant file-name search over a persisted in-memory index
- Hidden/system file coverage when Full Disk Access is granted
- Table, compact, two-pane preview, and icon grid layouts
- Everything-style query syntax: `ext:`, `size:`, `dm:`, `folder:`, `file:`, `path:`, `content:`, `tag:`, `type:`, negation, OR, quotes
- Finder workflows: Quick Look, rename, Move to Trash, Reveal in Finder, Get Info, tags, drag-out
- Dynamic external-volume indexing
- Custom index folders for NAS/SMB/network shares
- Folder-size indexing and Run Count sort
- `mvfind` CLI bundled inside the app
- `mv-mcp` MCP server bundled inside the app
- Lightweight in-app update checker against GitHub Releases
- GPL-3.0 source release

## First Run

For complete results, grant Full Disk Access:

```text
System Settings > Privacy & Security > Full Disk Access
```

Without Full Disk Access, Maverything still runs but macOS limits what it can read.

## Signing Status

This is an early public-test build. If this DMG is not yet Developer ID signed and notarized, macOS may show a Gatekeeper warning on first launch.

Use:

1. Right-click `Maverything.app`
2. Choose **Open**
3. Confirm once

## Checksum

Replace this after building the final DMG:

```text
SHA-256: <fill after ./make-dmg.sh>
```

## Known Limitations

- Full Disk Access is required for complete whole-disk results.
- Network shares may need scheduled rescans because they do not always deliver reliable file events.
- The updater checks and opens the latest DMG, but does not yet auto-install updates.
- Homebrew cask is still a draft.
